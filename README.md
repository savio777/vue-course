# vue-course

### steps to create project

- command to create: `npm create vue@latest vue-course`
- install tailwind: `npm install -D tailwindcss@3.3.5 postcss@latest autoprefixer@latest && npx tailwindcss init -p`
- install icons: `npm install primeicons`
- install json-server: `npm i json-server`
- install axios: `npm install axios`
- install vue-spinner: `npm install vue-spinner`
- install toast: `npm install vue-toastification@next`

### steps to run project

- run json-server: `npm run server`
- run vue project: `npm run dev`
